# Dump Filter

## Description

データを YAML 形式の文字列に変換するフィルターを作成します。

## Format

| Name  | Type   | Required | Description                                      |
| ----- | ------ | -------- | ------------------------------------------------ |
| class | string | yes      | プラグインの固有名称である `Dump` を指定します。 |
