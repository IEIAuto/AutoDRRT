# Precision Filter

## Description

データの小数点精度を指定するフィルターを作成します。

## Format

| Name   | Type   | Required | Description                                           |
| ------ | ------ | -------- | ----------------------------------------------------- |
| class  | string | yes      | プラグインの固有名称である `Precision` を指定します。 |
| digits | int    | yes      | 小数点精度を指定します。                              |
