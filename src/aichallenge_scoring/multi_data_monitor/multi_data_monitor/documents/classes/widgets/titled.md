# Simple Widget

## Description

タイトルの付けられたウィジェットを作成します。

## Format

| Name  | Type   | Required | Description                                                                |
| ----- | ------ | -------- | -------------------------------------------------------------------------- |
| class | string | yes      | プラグインの固有名称である `Titled` を指定します。                         |
| title | string | no       | データのタイトルを指定します。タイトルはデータの下に表示されます。         |
| notes | string | no       | パネルにカーソルを合わせたときに表示されるツールチップの内容を指定します。 |
