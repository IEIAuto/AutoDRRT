# Panel Stream

## Description

このストリームをユーザーが直接使用することはできません。
システム内部でウィジェットとストリームを接続するために使用されています。

## Format

| Name  | Type   | Required | Description                                              |
| ----- | ------ | -------- | -------------------------------------------------------- |
| class | string | yes      | プラグインの固有名称である `@panel` が指定されています。 |
